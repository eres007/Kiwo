TROUBLESHOOTING
════════════════

Q: Email not received after signup?
A: 1. Check spam/junk folder
   2. Verify RESEND_API_KEY is set correctly in Render
   3. Check Render logs for errors
   4. Wait a few minutes (emails can take time)

Q: How do I check if email was sent?
A: 1. Go to https://dashboard.render.com
   2. Select your service
   3. Click "Logs"
   4. Search for "Email sent via Resend"
   5. You'll see confirmation if email was sent

Q: Can I change the FROM email?
A: Yes, update FROM_EMAIL in Render environment variables
   Note: Must be verified in Resend dashboard first

Q: How do I verify a custom domain?
A: 1. Go to https://resend.com/domains
   2. Add your domain
   3. Follow DNS setup instructions
   4. Update FROM_EMAIL in Render
