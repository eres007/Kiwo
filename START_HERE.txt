WHAT IS KIWO?
═══════════════════════════════════════════════════════════════════════════════

Kiwo is an AI memory layer that remembers what matters across all your AI tools.

PROBLEM:
- Developers use multiple AI tools (Cursor, Claude Code, ChatGPT, Gemini)
- Each tool forgets everything from the others
- Developers re-explain their project 100+ times per week

SOLUTION:
- Captures context from one AI tool
- Refines it with AI (Qwen3)
- Stores it securely
- Injects it into the next AI tool automatically
- Developer never re-explains again

DEPLOYMENT IN 5 MINUTES
═══════════════════════════════════════════════════════════════════════════════

1. Go to https://vercel.com
2. Sign in with GitHub
3. Click "Add New..." → "Project"
4. Select: eres007/Kiwo
5. Root Directory: ./website
6. Environment Variable:
   - NEXT_PUBLIC_API_URL=https://kiwo.onrender.com
7. Click "Deploy"
8. Wait 2-3 minutes
9. Website live at: https://kiwo.vercel.app
