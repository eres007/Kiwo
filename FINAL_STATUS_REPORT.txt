HOW USERS USE KIWO
═══════════════════════════════════════════════════════════════════════════════

1. User visits website
2. User signs up with email
3. User logs in
4. User connects AI tools (Cursor, Claude Code, ChatGPT, etc.)
5. User tells AI tool about their project
6. Kiwo captures and stores the memory
7. User opens different AI tool
8. Kiwo injects relevant memories
9. New AI tool has full context
10. User never re-explains again
