WHAT IS KIWO?
═══════════════════════════════════════════════════════════════════════════════

Kiwo is an AI memory layer that remembers what matters across all your AI tools.

Problem: You use Cursor, Claude Code, ChatGPT, and Gemini. Each one forgets
         everything you told the others. You re-explain your project 100 times.

Solution: Kiwo captures context from one AI tool and injects it into the next.
          Your AI tools now remember your project, preferences, and decisions.

DEPLOY WEBSITE IN 5 MINUTES
═══════════════════════════════════════════════════════════════════════════════

1. Go to https://vercel.com
2. Sign in with GitHub
3. Click "Add New..." → "Project"
4. Select repository: eres007/Kiwo
5. Configure:
   - Root Directory: ./website
   - Environment Variable:
     * Name: NEXT_PUBLIC_API_URL
     * Value: https://kiwo.onrender.com
6. Click "Deploy"
7. Wait 2-3 minutes
8. Website live at: https://kiwo.vercel.app

Done! ✅

API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

PUBLIC (No JWT Required):
POST   /api/auth/signup              - Create account
POST   /api/auth/login               - Login
GET    /health                       - Health check

PROTECTED (JWT Required):
POST   /api/memory/capture           - Capture memory
GET    /api/memory/retrieve          - Get memories
GET    /api/memory/list              - List all memories
PUT    /api/memory/:id               - Update memory
DELETE /api/memory/:id               - Delete memory
GET    /api/memory/search            - Search memories
