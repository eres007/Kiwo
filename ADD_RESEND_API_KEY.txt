═══════════════════════════════════════════════════════════════════════════════
TROUBLESHOOTING
════════════════

Problem: Email not received
Solution: 
1. Check spam/junk folder
2. Wait 5 minutes (emails can be slow)
3. Check Render logs for errors
4. Verify API key is correct

Problem: Render deployment failed
Solution:
1. Check Render logs for error message
2. Verify API key format is correct
3. Try redeploying manually

Problem: Email shows "onboarding@resend.dev" as sender
Solution:
This is normal for free tier. To use custom domain:
1. Verify domain in Resend dashboard
2. Update FROM_EMAIL in Render
3. Redeploy
